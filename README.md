# Console-based-Exam-system
The Smart Exam System is a command-line-based multiple-choice quiz program developed in MIPS Assembly
Language and executed using the MARS (MIPS Assembler and Runtime Simulator). It mimics a basic exam
environment where users must log in with credentials and then select a subject to answer 10 quiz questions.
This project demonstrates the use of assembly programming in real-world scenarios, such as exam systems and
user authentication.
